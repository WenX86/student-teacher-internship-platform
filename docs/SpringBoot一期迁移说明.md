# Spring Boot 一期迁移说明

## 迁移目标

将已跑通的一期 Node 原型后端，迁移为需求文档推荐的正式技术栈后端：

- Spring Boot
- Spring Security
- MyBatis-Plus
- MySQL

当前阶段先使用 H2 内存库承载一期演示数据，确保业务规则和接口形态先稳定，再平滑切换到 MySQL。

## 一期已落地范围

- 登录认证与角色权限控制
- 学生、教师、学院基础管理
- 实习单位信息库
- 指导教师申请、教师确认、学院复核
- 实习申请提交与学院审批
- 任课实习、班主任实习核心表单
- 教师审核、学院终审归档
- 消息提醒与基础统计

## 当前验证结果

已完成以下验证：

- Maven `compile` 成功
- Maven `package` 成功
- Spring Boot 启动成功
- `/api/health` 可访问
- `college01 / 123456` 登录成功
- 学院端 `/api/dashboard`、`/api/students` 可返回数据
- 学生端 `/api/mentor-applications`、`/api/forms` 可返回数据

## 当前工程价值

这版后端已经不是纯原型，而是一期正式后端底座，具备：

- 可编译
- 可打包
- 可启动
- 可联调
- 可继续按模块扩展

## 下一步建议

建议按下面顺序继续推进：

1. 输出一期接口清单与字段说明

4. 增加基础单元测试和接口测试
