# 一期接口清单与字段说明

## 基本说明

- 基础地址：`http://localhost:8080/api`
- 认证方式：除 `/auth/login`、`/health` 外，均需 `Authorization: Bearer <token>`
- 返回结构：

```json
{
  "success": true,
  "data": {}
}
```

## 认证与账户

### POST `/auth/login`

- 说明：账号密码登录
- 请求体：
  - `account`：登录账号
  - `password`：登录密码
- 返回字段：
  - `token`：访问令牌
  - `user.id`
  - `user.account`
  - `user.name`
  - `user.role`
  - `user.collegeId`
  - `user.mustChangePassword`
  - `user.lastLoginAt`
  - `user.profile`

### GET `/auth/me`

- 说明：获取当前登录用户信息

### POST `/auth/change-password`

- 说明：修改密码
- 请求体：
  - `newPassword`

### GET `/auth/login-records`

- 说明：获取当前用户登录日志

## 公共能力

### GET `/health`

- 说明：健康检查

### GET `/dashboard`

- 说明：按角色返回首页看板数据
- 角色差异：
  - 学生：`internshipStatus`、`mentorStatus`、`todoCount`、`archivedCount`
  - 教师：`studentCount`、`pendingMentorRequests`、`pendingReviewCount`、`archivedCount`、`completionRate`
  - 学院管理员：`studentCount`、`teacherCount`、`pendingMentorReviewCount`、`pendingInternshipReviewCount`、`pendingArchiveCount`、`riskStudentCount`
  - 超级管理员：`activeUsers`、`collegeApplicationCount`、`pendingCollegeApplicationCount`、`totalForms`、`unreadMessages`

### GET `/messages`

- 说明：获取当前用户消息列表
- 返回字段：
  - `id`
  - `type`
  - `title`
  - `content`
  - `link`
  - `read`
  - `createdAt`

### POST `/messages/{id}/read`

- 说明：消息标记已读

## 学院管理员端

### GET `/students`

- 说明：获取本学院学生列表
- 返回字段：
  - `id`
  - `name`
  - `studentNo`
  - `major`
  - `className`
  - `phone`
  - `internshipType`
  - `internshipStatus`
  - `account`
  - `accountStatus`
  - `mentorStatus`
  - `internshipApplicationStatus`

### POST `/students`

- 说明：新增学生
- 请求体：
  - `name`
  - `studentNo`
  - `major`
  - `className`
  - `phone`
  - `internshipType`

### POST `/students/{id}/reset-password`

- 说明：重置学生密码为 `123456`

### PATCH `/students/{id}/status`

- 说明：启用/停用学生账号
- 请求体：
  - `status`：`ACTIVE` 或 `DISABLED`

### GET `/teachers`

- 说明：获取教师列表
- 返回字段：
  - `id`
  - `name`
  - `employeeNo`
  - `department`
  - `phone`
  - `account`
  - `accountStatus`
  - `studentCount`

### POST `/teachers`

- 说明：新增教师
- 请求体：
  - `name`
  - `employeeNo`
  - `department`
  - `phone`

### GET `/organizations`

- 说明：获取实习单位列表

### POST `/organizations`

- 说明：新增实习单位
- 请求体：
  - `name`
  - `address`
  - `contactName`
  - `contactPhone`
  - `nature`
  - `cooperationStatus`

### PATCH `/organizations/{id}`

- 说明：更新实习单位
- 请求体与新增一致

### GET `/reports/summary`

- 说明：获取一期基础统计
- 返回结构：
  - `students.total`
  - `students.applied`
  - `students.active`
  - `teachers.total`
  - `teachers.activeGuidanceCount`
  - `forms.total`
  - `forms.archived`
  - `forms.rejected`
  - `forms.archiveRate`
