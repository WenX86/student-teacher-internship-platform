# 师范生教育实习全过程管理平台使用文档

## 1. 文档说明

本文档为系统平台简略使用文档，面向学生、指导教师、学院管理员和超级管理员，帮助用户快速了解平台登录方式、主要功能入口和常用操作流程。

## 2. 平台简介

师范生教育实习全过程管理平台用于支撑教育实习全过程线上管理，覆盖指导教师申请、实习申请、过程表单填报、审核归档、指导记录、评价确认、消息提醒和统计分析等业务。

## 3. 登录说明

### 3.1 登录入口

用户进入系统后，访问登录页，输入账号和密码即可进入对应角色门户。

### 3.2 账号规则

系统支持以下四类角色登录。

1. 学生
2. 指导教师
3. 学院管理员
4. 超级管理员

3. 不同角色登录后会自动进入对应门户首页。

## 5. 学生端使用说明

### 5.1 首页看板

学生登录后默认进入首页，可查看当前指导关系状态、实习状态、待办数量和归档情况。

### 5.2 指导教师申请

操作步骤如下。

1. 进入“指导教师申请”页面。
2. 选择申请教师并填写备注信息。
3. 提交申请后，等待教师确认。
4. 教师确认后，等待学院管理员复核。
5. 审核通过后，指导关系生效。

### 5.3 实习申请

操作步骤如下。

1. 进入“实习申请”页面。
2. 选择实习单位，填写岗位、实习时间、成绩目标和备注。
3. 按需上传附件后提交申请。
4. 等待学院管理员审批。
5. 审批通过后，学生进入实习中状态。

注意：只有指导关系生效后，才能提交实习申请。

### 5.4 表单填报

操作步骤如下。

1. 进入“表单中心”页面。
2. 选择对应模板新建表单。
3. 填写表单内容，可先保存为草稿。
4. 确认无误后提交审核。
5. 若被退回，可修改后再次提交。
6. 审核通过后，表单进入归档状态。
7. 如需修改已归档表单，先提交修改申请，等待学院审批通过后再重新编辑并提交。

### 5.5 消息与结果查看

1. 在“消息中心”查看待办提醒、退回通知和审核结果。
2. 在“结果查看”页面查看评价结果、归档结果和最终成绩。

## 6. 指导教师端使用说明

### 6.1 指导申请处理

操作步骤如下。

1. 进入“指导申请处理”页面。
2. 查看学生提交的指导申请。
3. 根据情况选择“通过”或“驳回”。
4. 填写审核意见并提交。

### 6.2 材料审核

操作步骤如下。

1. 进入“材料审核”页面。
2. 查看学生提交的过程表单。
3. 审核表单内容与附件。
4. 填写评分和审核意见。
5. 选择“通过”或“退回”。

### 6.3 指导记录维护

操作步骤如下。

1. 进入“指导记录”页面。
2. 选择学生并填写指导时间、指导方式、存在问题、指导建议和后续跟进。
3. 保存记录后，系统留存为过程管理依据。

### 6.4 评价管理

操作步骤如下。

1. 进入“评价管理”页面。
2. 选择学生并填写阶段评语、总结评语和成绩。
3. 提交评价后，等待学院确认。

### 6.5 预警与消息

1. 在“预警提醒”页面查看超时待审事项。
2. 在“消息中心”查看系统通知和流程结果。

## 7. 学院管理员端使用说明

### 7.1 学生与教师管理

1. 在“学生管理”页面新增学生、重置密码、启停账号。
2. 在“教师管理”页面新增教师、重置密码、启停账号。

### 7.2 指导关系复核

操作步骤如下。

1. 进入“指导关系”页面。
2. 查看教师已确认的申请记录。
3. 进行复核并选择通过或退回。
4. 复核通过后，指导关系正式生效。

### 7.3 实习单位管理

1. 进入“实习单位管理”页面。
2. 维护单位名称、地址、联系人、联系电话、单位性质和合作状态。

### 7.4 实习申请审批

操作步骤如下。

1. 进入“实习申请审批”页面。
2. 查看学生提交的申请内容。
3. 填写单位确认情况、审批意见和结果。
4. 提交审批结论。

### 7.5 归档中心

操作步骤如下。

1. 进入“归档中心”页面。
2. 查看待学院处理的学生表单。
3. 选择单条归档或批量归档。
4. 如材料不合格，可退回学生修改。
5. 对已归档后的修改申请，可在此页面审批通过或驳回。

### 7.6 评价确认

操作步骤如下。

1. 进入“评价确认”页面。
2. 查看教师提交的评价记录。
3. 填写学院意见和学院评分。
4. 完成确认后，评价结果正式生效。

### 7.7 报表与预警

1. 在“报表中心”查看学生、教师、学院维度统计信息。
2. 在“预警中心”查看超时未审、超时未改和待确认记录。
3. 对需要督办的事项可发送提醒消息。

## 8. 超级管理员端使用说明

### 8.1 基础数据管理

维护本校学院信息、角色和平台基础数据。

### 8.2 参数配置

维护超时天数、预警等级和消息模板等系统参数。

### 8.3 表单模板管理

操作步骤如下。

1. 进入“表单模板管理”页面。
2. 新增或修改模板编码、名称、分类、描述和适用类型。
3. 配置模板字段结构并设置是否启用。

### 8.4 日志审计

在“日志审计”页面查看登录日志、审核操作日志和系统关键操作记录。

## 9. 常用业务流程

### 9.1 学生实习业务主流程

1. 学生登录系统。
2. 提交指导教师申请。
3. 教师确认申请。
4. 学院复核指导关系。
5. 学生提交实习申请。
6. 学院审批实习申请。
7. 学生填报过程表单。
8. 教师审核表单。
9. 学院归档表单。
10. 教师提交评价。
11. 学院确认评价。

### 9.2 表单状态说明

当前表单主要状态包括。

1. 草稿
2. 教师审核中
3. 教师退回
4. 学院审核中
5. 学院退回
6. 已归档
7. 修改申请中
8. 允许修改

### 9.3 账号状态说明

账号状态包括。

1. `ACTIVE`：可正常登录。
2. `DISABLED`：不可登录。

## 10. 常见问题

### 10.1 无法登录系统怎么办

1. 确认账号和密码是否输入正确。
2. 确认账号是否被停用。
3. 若为重置密码后首次登录，请使用默认密码后尽快修改。

### 10.2 为什么无法提交实习申请

通常是因为指导教师申请尚未完成学院复核，指导关系未正式生效。

### 10.3 为什么表单不能修改

只有草稿、教师退回、学院退回、允许修改状态下的表单允许学生继续修改；已归档表单需要先提交修改申请并等待学院审批。

### 10.4 为什么教师不能审核某学生材料

教师只能处理与自己存在有效指导关系的学生材料。

## 11. 使用建议

1. 学生应先完成指导教师申请，再进行实习申请与材料填报。
2. 教师和学院管理员应及时处理待办，避免超时预警。
3. 所有用户应及时查看消息中心，避免遗漏退回和审核结果通知。
4. 管理员应定期检查账号状态、系统参数和模板启用情况。
